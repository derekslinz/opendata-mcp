This PR adds 4 new high-quality, key-less providers to the OpenData MCP registry:
- **DBnomics**: Economic data aggregator.
- **PubChem**: Chemical database.
- **RCSB PDB**: Biological structure metadata.
- **NOAA Aviation Weather**: Real-time aviation weather data.

Changes:
- Added 4 new provider modules in `opendata_mcp/providers/`.
- Updated `registry.py` with new providers and domains ('biology', 'chemistry').
- Updated `README.md` count and tables.
- Added unit tests for each new provider.
- Verified SSE transport functionality.
- Removed legacy ARM placeholder.
